Main

  • How to manage marks
    • You can add a new mark by enter the frame number of the new marks in

      input field[2]

      , and click the "+" button. The new mark will appear in the

      mark list [3]

    • You can change a mark by drag the corresponding cursor in

      range selector [1]

      , or change the corresponding input field in

      mark list [3]

    • You can delete a mark by click "-" for the corresponding input field in

      mark list [3]

  • How to set weights
    1. Choose the reference you want to set weights in the

      drop down list[0]

      , The information of the selected reference will be show in

      range selector [1]

    2. Click the interval that you want to set weights in the

      range selector [1]

      , the selected interval will have a

      background color

      . Weights of the selected interval will be show in the

      weights input fields [9]

    3. Change the weight number on the joint you want in the

      weights input fields [9]

  • How to set student recording
    1. Upload the student recording with the

      file input field [4]

      . After loading, the information of uploaded recording will be show in the

      range selector [5]

      .
    2. Select the range by drag the cursors in

      range selector [5]

      or change the frame number in

      input fields [6]

      . Unselected ranges will be dim.
  • Get results
    1. Set reference and set student recording.
    2. Do animation settings with

      switches [7]

      .
    3. Click

      "submit" button [8]

      and wait until jump to the result page.
  • How to go management page
    • Click the

      "Management page" button [10]

  • How to show/hide descriptions
    • turn the

      switch [11]

      on/of.

Result

  • Explain of "result.xlsx"
    • Sheets in "result.xlsx" consist of columns representing different joints and rows corresponding to frame numbers. There are six sheets:
    1. "Segment Position" is the position distances between expert and learner for each joint in each frame. The unit of measurement is m
    2. "Segment Velocity" is the Euclidean velocity distance between expert and learner for each joint in each frame. The unit of measurement is m/s
    3. "Segment Acceleration" is the Euclidean acceleration distance between expert and learner for each joint in each frame. The unit of measurement is (m/s)^2
    4. "Segment Angular Velocity" is the Euclidean angular velocity distance between expert and learner for each joint in each frame. The unit of measurement is deg/s
    5. "Segment Angular Acceleration" is the Euclidean angular acceleration distance between expert and learner for each joint in each frame. The unit of measurement is (deg/s)^2
