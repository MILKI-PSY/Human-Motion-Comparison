    1. Click the interval that you want to set weights in the

      range selector [1]

      background color

      . Weights of the selected interval will be show in the

      weights input fields [9]

      weights input fields [9]

      range selector [5]

      range selector [5]

      input fields [6]

Result

  • Explain of animation
    • The result of the comparison will be present as an animation.
    • There are

      three frame numbers [0]

      . The first one corresponds to the frame number of the resulting animation, the second one corresponds to the frame number of the reference from its original recording, and the third one corresponds to the frame number of the student recording from its original recording.
    • The

      legend [1]

      can help you understand the correspondence between characters and colors.
    • The

      heat map [2]

      reflects the real-time score, the higher the score, the redder the color.
  • How to download "result.xlsx"
      • Click

      "Download Comparison Result" [3]

  • Explain of "result.xlsx"
    • Sheets in "result.xlsx" consist of columns representing different joints and rows corresponding to frame numbers. There are six sheets:
    1. "Segment Position" is the position distances between expert and learner for each joint in each frame. The unit of measurement is \( m \)
    2. "Segment Velocity" is the Euclidean velocity distance between expert and learner for each joint in each frame. The unit of measurement is \( m/s \)
    3. "Segment Acceleration" is the Euclidean acceleration distance between expert and learner for each joint in each frame. The unit of measurement is \( (m/s)^2 \)
    4. "Segment Angular Velocity" is the Euclidean angular velocity distance between expert and learner for each joint in each frame. The unit of measurement is \( deg/s \)
    5. "Segment Angular Acceleration" is the Euclidean angular acceleration distance between expert and learner for each joint in each frame. The unit of measurement is \( (deg/s)^2 \)
    6. “Score”: It is a number to evaluate the performance of each joint in each frame. \[Score = {Euc(w_0 - w_1) \over max(w_0, c) }. Weight\] Where \( c \) is a hyperparameter that can be set in the program, and the default number of c is 1
  • Explain of average score image
    • The

      average score image [4]

      is a heat map of average score of whole comparison, it can be understood as an overall score. The higher the score, the redder the color.
